# Algorithms & Data Structures
Solutions for coding interview challenges.

## Commands
Run these commands from the root directory:

### Run Project

```bash
# make run
dotnet run --project src/cracking.csproj
```

### Run Tests

```bash
#make test
dotnet test
```

## Structure
*   `src/` — Implementation of algorithms.
*   `tests/` — xUnit test suites.
*   `Cracking.slnx` — Solution file.

## Tech Stack
*   .NET 10.0
*   C# 14
*   xUnit
